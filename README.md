# Proyecto c12z

Un blog/web personal que voy a ir construyendo poco a poco.

## estructura del proyecto

las carpetas principales son:

- assets: donde están todas las fuentes y los recursos visuales: imágenes, videos, gifs...
- components: donde están todos los elementos que construyen la app
- global: información común para toda la app
- interfaces: donde están todas las interfaces (esta la tengo que eliminar)
- utils: utilidades que pueden ser útiles para la app como formateos de fecha
- styles: donde están recogidos los componentes de css generales

carpetas obligatorias en astro:

- content: capeta de astro donde está el contenido
- layout: donde están las UIs de las páginas
- pages: donde están las rutas de las páginas
